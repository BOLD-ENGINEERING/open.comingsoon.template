# Open Coming-Soon Template

A coming soon template built for pre-project announcements with public repository activity tracking.

## Tech Stack

- Vue 3 + TypeScript + Vite (Rolldown)
- Tailwind v4
- PrimeVue
- Pinia

## Requirements

- Node >= 24.14.1
- Bun
